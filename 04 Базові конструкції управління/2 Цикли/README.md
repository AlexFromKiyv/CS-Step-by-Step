# For

```cs
UsingFor();
static void UsingFor()
{
	int length = 10;
	for (int i = 0; i < length; i++)
	{
		Console.WriteLine(i);
	}

	//Console.WriteLine(i); //does not exit in context
}
```
Якщо відомо яку фіксовану кількість ітерацій треба зробити for швидкий і простий варіант.
