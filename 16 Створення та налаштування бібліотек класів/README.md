# Створення та налаштування бібліотек класів

У більшості прикладів, наведених у цій книзі, ви створювали «автономні» виконувані програми, в яких вся логіка програмування була упакована в одну збірку (*.dll) та виконувалася за допомогою dotnet.exe (або копії dotnet.exe, названої на честь збірки). Ці збірки використовували лише базові бібліотеки класів .NET. Хоча деякі прості програми .NET можуть бути створені з використанням лише базових бібліотек класів, цілком ймовірно, що ви (або ваші колеги) будете зазвичай ізолювати логіку програмування повторного використання в користувацькі бібліотеки класів (файли *.dll), які можна використовувати спільно між програмами.

У цьому розділі ви почнете з вивчення деталей розділення типів на простори імен .NET. Після цього ви детально розглянете бібліотеки класів у .NET, різницю між .NET та .NET Standard, конфігурацію застосунків, публікацію консольних застосунків .NET та пакування бібліотек у пакети NuGet, які можна використовувати повторно.
